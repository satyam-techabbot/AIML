# Generative AI

* Generative AI is a type of AI that **creates new content** such as:
  * Text
  * Images
  * Audio
  * Video
  * Code
* It learns patterns from existing data and generates **human-like outputs** 

Key idea:
> **Not just analyzing → but creating**

---

## Generative AI vs Traditional AI
| Type                            | Function                               |
| ------------------------------- | -------------------------------------- |
| Traditional (Discriminative AI) | Classifies / predicts (spam detection) |
| Generative AI                   | Creates new data (text, images, etc.)  |

Generative AI produces **original outputs**, not just labels 

---

## How Generative AI Works

### Step 1: Training
* Trained on **large datasets** (text, images, etc.)
* Learns patterns using deep learning

### Step 2: Learning
* Models learn parameters (millions/billions)

### Step 3: Generation
* Takes a **prompt**
* Generates output based on learned patterns 

---

## Types of Generative AI (by output)

### Text
* Uses **LLMs (Large Language Models)**
* Predicts next word/token

### Images
* Uses **Diffusion Models**
* Converts noise → realistic image

### Audio
* Text-to-speech systems

### Video
* Multimodal models generate sequences 

---

## Key Concepts in GenAI

### 1. Transformers / Autoregressive Models
* Generate text step-by-step
* Use **self-attention**
* Example: GPT

---

### 2. Diffusion Models
* Start with random noise
* Gradually refine into image/audio

---

### 3. Fine-Tuning
* Adapting pre-trained models to specific tasks

Techniques:
* LoRA
* QLoRA
* PEFT
* RLHF (Reinforcement Learning from Human Feedback)

---

### 4. RAG (Retrieval-Augmented Generation)
* Combines:
  * Retrieval (search data)
  * Generation (LLM output)

Process:
1. Store data in vector DB
2. Retrieve relevant info
3. Add to prompt
4. Generate grounded response 

---

### 5. AI Agents
* Autonomous systems using LLMs
* Can:
  * Plan
  * Reason
  * Use tools/APIs 

---

## Advantages
* Faster content creation
* Personalization

---

## Challenges / Limitations
* Bias in outputs
* Hallucinations (wrong info)
* Ethical concerns
* Dependency on training data

---

## Human + AI Relationship
* GenAI **augments humans**, not replaces them
* Helps in:
  * Creativity
  * Decision making
  * Automation
