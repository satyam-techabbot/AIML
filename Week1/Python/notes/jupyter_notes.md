# Jupyter

### What is Jupyter notebook?
Jupyter Notebook is an open-source, interactive web application used to create and share documents containing live code, equations, visualizations, and narrative text.

### Key Features and Capabilities
- Interactive Computing: Users can run code in chunks (cells), making it easy to test, debug, and iterate on data analysis projects without restarting the entire program.
- Rich Documentation: Notebooks can blend executable code with formatted text (Markdown), images, LaTeX mathematical formulas, and interactive visualizations.

- "Notebook" Format: Files are saved with an .ipynb extension and can be converted to formats like HTML, PDF, or Python scripts.
- Web-Based Interface
- Kernels: Independent processes (kernels) handle the execution of code in specific languages (e.g., Python, R). 

### Common Use Cases
- Data cleaning and transformation.
- Numerical simulation and statistical modeling.
- Data visualization.
- Machine learning development. 

### Installation
